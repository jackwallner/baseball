## Metrics Calculable from Play-by-Play Data (Non-Statcast)

These metrics **CAN** be calculated from box score/play-by-play data (Retrosheet, FanGraphs, Baseball-Reference) and do NOT require Statcast tracking:

| Metric | Calculable Since | Data Source |
|--------|------------------|-------------|
| **K%** (Strikeout Rate) | 1900s+ | Box scores (SO / PA) |
| **BB%** (Walk Rate) | 1900s+ | Box scores (BB / PA) |
| **K/BB** Ratio | 1900s+ | Box scores |
| **AVG** | 1900s+ | Box scores (H / AB) |
| **OBP** | 1900s+ | Box scores ((H+BB+HBP) / PA) |
| **SLG** | 1900s+ | Box scores (TB / AB) |
| **OPS** | 1900s+ | Calculated from OBP+SLG |
| **GB%** (Ground Ball %) | ~2002+ | Play-by-play string parsing |
| **FB%** (Fly Ball %) | ~2002+ | Play-by-play string parsing |
| **LD%** (Line Drive %) | ~2002+ | Play-by-play string parsing (subjective) |
| **IFFB%** (Infield Fly %) | ~2002+ | Play-by-play string parsing |
| **HR/FB** | ~2002+ | Home runs / Fly balls |
| **BABIP** | 1900s+ | (H-HR) / (AB-K-HR+SF) |
| **wOBA** | 1900s+ | Linear weights from play outcomes |
| **wRC+** | 1900s+ | Park-adjusted from wOBA |
| **Clutch** | 1974+ | Win Probability Added |
| **WPA/LI** | 1974+ | Context-neutral WPA |

### What This Means for Backfill

**You CAN get K%, BB%, basic rate stats for ALL years (2015-2026)** by:
1. Querying FanGraphs API or baseballstats endpoints
2. Parsing Retrosheet play-by-play data
3. Scraping Baseball-Reference

**The audit showing K% and BB% as "missing" is misleading** - they may not be in the Statcast tables, but they're absolutely available from traditional box score data.

## Key Takeaway

**Maximum possible completeness by year:**
- 2015: ~25% (7/28 metrics)
- 2016: ~54% (15/28 metrics)
- 2017-2019: ~57% (16/28 metrics)
- 2020: ~71% (20/28 metrics)
- 2021-2022: ~75% (21/28 metrics)
- 2023: ~96% (27/28 metrics, some partial)
- 2024+: 100% (28/28 metrics)

The audit results showing ~50% completeness for 2015-2019 are actually **at or near the maximum possible** given when these metrics were first tracked.
